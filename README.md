# Habit Tracker

Трекер привычек - простое приложение для отслеживания ежедневных привычек.

## Возможности

- ✅ Добавление и удаление привычек
- ✅ Отметка выполнения за каждый день
- ✅ Календарь последних 7 дней
- ✅ Подсчет серии дней подряд
- ✅ Общая статистика
- ✅ Сохранение данных локально

## Установка

APK файл доступен в разделе Actions → Artifacts

## Технологии

- React + Vite
- Tailwind CSS
- Capacitor (для Android)
- GitHub Actions (автоматическая сборка)
